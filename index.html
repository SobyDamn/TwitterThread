About

Twitter has many great people sharing valuable information in the form of threads.
This is a twitter bot which can be used to compile threads and view it at one place or share it with others, this can also be used to print the thread and save it as pdf format for later consumption.
In case you're more interested in the working you can check my GitHub repository and you can contact me in case any help needed.

How to use?

There are two methods in which a thread can be compiled for you to view.

#Method 1

  1. Go to the thread.

  2. At any part of the thread reply with @ThreaderTweet compile.
    The bot will soon reply to you with to a link of compiled thread.

#Method 2

  1. Go to the bottom of the thread i.e the last tweet of the thread.

  2. At the bottom reply with @ThreaderTweet.
    The bot will soon reply to you with to a link of compiled thread.

NOTE:-Try the 2nd method in case first one fails.

Help

  1. Getting thread too old Error!
  2. Not receiving any reply.
  3. How to host my own twitter bot?
  4. Received link not working.
  5. Compiled thread is incorrect or doesn't contain all the tweets.

1.Getting thread too old Error!

This might happen if the thread is too old, this is because of limitation of twitter API. At the current moment I have limited the bot from looking every tweets to save time.
To solve this try the second method, which is go to the bottom of the thread and call the bot without any command check here.

2.Not receiving any reply.

This can happen because of multiple reasons.

  • Thread might be too old and long and it's taking time to process it, try waiting for few more minutes.
  • Bot might be turned off for maintenance or not active anymore. You can contact me to get more information regarding this.

3.How to host my own twitter bot?

I will be a writing a blog regarding this in the meantime check my GitHub repository.

4.Received link not working.

This might be because of some technical error and the thread being removed from the site,try compiling the thread again.
Sorry for inconvenience.

5.Compiled thread is incorrect or doesn't contain all the tweets.

In case the compiled thread is incorrect or incomplete, try using 2nd method to compile.
This can happen if one or more tweets of the thread is not visible for the public or is pending for verification by twitter.
